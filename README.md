# 🛒 E-Commerce API Testing Portfolio

A comprehensive Manual API Testing portfolio demonstrating real-world testing of E-Commerce business workflows using Postman. This repository showcases test planning, test design, API validation, defect management, execution reporting, and quality assurance best practices across Authentication, Products, Categories, Cart Management, User Management, Search, and Pagination APIs.

---

## 🎯 Project Overview

Modern e-commerce platforms rely heavily on APIs to manage products, customers, shopping carts, authentication, and business transactions.

This project demonstrates how a QA Engineer validates critical API functionality to ensure:

✔️ APIs return accurate and consistent responses

✔️ Business workflows function as expected

✔️ Error handling and validations are properly implemented

✔️ Status codes and response headers comply with API standards

✔️ Authentication and authorization mechanisms work correctly

✔️ Search and pagination functionality deliver reliable results
